Action Title
============
让你的网站标题滚动起来。

基本用法
----------
    $.ActionTitle.show('This is a ACTION title!');
      
    $.ActionTitle.clear();
    
如果不使用 JQuery 的话:

    ActionTitle.show('This is a ACTION title!');

高级用法
---------
    $.ActionTitle.show({
    	prefix: 'Fixed ',
    	text: 'This is a ACTION title!',
    	fps: 6
    });
